## Summary

This paper reports a statistically robust North–South asymmetry of the heavy-tail index of the NIR-H flux distribution (Sersic H-band flux) in the Euclid Multi-Epoch Release Q1 public catalogue, accessed via ESASky 7.40.

Using the Hill estimator with threshold **k = 50**, declared prior to any analysis, applied to five independent sky zones, we measure a block-averaged contrast **Δα ≈ 0.80** between North zones (ā_N ≈ 2.01) and South zones (ā_S ≈ 1.21).

The contrast persists under all robustness tests: k-stability, bootstrap (P(Δα > 0) ≈ 0.996), leave-one-out (0.67 ≤ Δα ≤ 0.93), KS test (p < 10⁻⁶), and variance inflation (Z > 1.6 with tripled variance).

No cosmological interpretation is imposed. The result is conservatively interpreted as a **probable large-scale dipole precursor**, testable with Euclid Q2/Q3 data expected in **October 2026**.
